# Secrets handling

## How to: generate a secret

1. Edit the file "secret.nix" to add the secret and select the decryption hosts

2. Run the command
```bash
agenix -e "name_of_the_secret_
```

3. Insert the password

4. Use it!

## How to: reencrypt all the secrets

1. Run the command
```bash
agenix --rekey
```
